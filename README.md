# Backend

## Prerequisitos

Para una mejor organizacion se recomienda crear un virtual environment con el comando:

```bash
python3 -m venv venv
```

Luego corremos el virtual environment con:

```bash
source venv/bin/activate
```

Ahora necesitamos instalar todas las librerias que usa la aplicaci√≥n usando el comando:

```bash
pip3 install -r requirements.txt
```

## Uso

Para correr el servidor usar el comando:

```bash
python main.py
```

Para correr los tests:

```bash
pytest tests
```
